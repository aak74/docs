Документация для большинства проектов отсутствует

Распространение знаний только через личное общение. Большой объем знаний только в одной голове.

Для крупных проектов

Документация

  • Пользователя
  • Администратора
  • Развертывание на проде и эксплуатация
  • Развертывание для разработчика
  • Архитектурные решения
Для крупных проектов есть хорошая документация пользователя и администратора. Есть документация по развертыванию системы. Представляет собой документ формата docx на десяток страниц. И не очень внятная документация разработчика. Процесс развертывания очень часто непростой. Подключение нового разработчика в проект дорого.

Обратный закон Конвея

При разработке огромного для нас проекта пришлось внедрять новые подходы к разработке включая и документацию. В этом проекте часть документации была написана ДО старта разработки.
